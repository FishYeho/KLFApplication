Key Features of the Website

This website contains three modules as per the specifications of the assignment:

1. The User Management Page

The user management page was created in order to be able to add new users to the website's database, as well as to modify the fields of the site's existing users. This page can be accessed by clicking on the "Manage Users" button located in the Navbar.

2. The Home Page

This home page was meant to be the landing page of the website and needed to incorporate the following three modules. First, it needed a Navbar which can be seen at the top of this page. Second, it was required to have a contact form, available at the bottom of this page. And lastly, it needed to have a login form, which I decided to make an individual page in order to allow for the proper separation of concerns. The login form can be accessed by clicking on the "login" button located in the Navbar.

3. The Reports Page

This reports page was required to produce a report that would display which users have been active on the website, as well as what activities they were performing. Unfortunately, the reports page is still in development and will not be available in this version of the assignment.
